# unityVR-project

VR项目，推荐使用 Unity 2022.3.62f2c1 编辑器版本。

## 克隆项目

本项目使用 Git LFS 管理贴图、音频、模型等大文件。请不要使用 GitHub 页面上的 Download ZIP，否则可能缺少资源文件。

克隆前请先安装 Git LFS：https://git-lfs.com

```bash
git lfs install
git clone https://github.com/Roins-hub/unityVR-project.git
cd unityVR-project
git lfs pull
```

克隆完成后，使用 Unity 2022.3.62f2c1 打开项目文件夹。
